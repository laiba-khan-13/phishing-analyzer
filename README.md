#  Phishing Email Analyzer

A simple web tool to analyze suspicious emails and detect potential phishing attempts.

##  Live Demo

**Try it here:** [https://laiba-khan-13.github.io/phishing-analyzer](https://laiba-khan-13.github.io/phishing-analyzer)

## What It Does

This tool checks emails for common phishing red flags:

-  **Suspicious sender domains** (typos like `amaz0n`, `paypa1`)
-  **Urgent/scary language** ("URGENT", "account suspended", "act now")
-  **Requests for personal info** (passwords, credit cards, SSN)
-  **Suspicious links** (shortened URLs, HTTP instead of HTTPS)
-  **Poor grammar & generic greetings** ("Dear Customer")
-  **Threats** ("legal action", "account closed")
-  **Dangerous attachments** (.zip, .exe files)

##  How to Use

1. Open the [live website](https://laiba-khan-13.github.io/phishing-analyzer)
2. Paste a suspicious email into the text box
3. Click **"Analyze Email"**
4. Get an instant risk score and detailed report!
